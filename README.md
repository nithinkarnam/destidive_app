# Destination Services App

## Overview

The Destination Services App is a web application that allows users to explore and discover attractions available in different destinations. Users can select a destination, submit their choice, and view a list of attractions with ratings in that destination.

## Features

- **Destination Selection:** Users can choose a destination from the available options.
- **Service List:** The app displays a list of attractions(as services) with their ratings in the selected destination.
- **Responsive Design:** The app is designed to work seamlessly on both desktop and mobile devices.

## Technologies Used

- React.js
- Bootstrap
- React Spring (for animations)
